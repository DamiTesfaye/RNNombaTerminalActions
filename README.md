# rn-nomba-terminal-actions

React Native module for Nomba terminal actions

## Installation

```sh
npm install rn-nomba-terminal-actions
```

## Usage

```js
import { multiply } from 'rn-nomba-terminal-actions';

// ...

const result = await multiply(3, 7);
```
